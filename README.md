# svgMap
svgMap是一个基于Raphael的svg地图组件，可以兼容ie6+现代浏览器，在api的设置上比较简单，目的就是要做一款比较小巧的组件，同时又能满足基本的地图展示需求，在加载速度和性能上都不比Canvas地图差。
所以广泛适用于基本的地图数据展示页面。

##  文档与示例

[Demo](http://rockyuse.github.io/widget/svgmap/svgmap.html)

##  兼容性

测试通过：IE6+、Chrome、Firefox、Safari、Opera
